# 🌿 Plant Tracker Client

**Plant Tracker** is a sleek and user-friendly web app that helps users manage and monitor their personal plant collection. From tracking care instructions to managing plant data, this application keeps everything organized for plant lovers.

---

## 🚀 Main Features

✅ **User Authentication** – Secure login and signup using Firebase Auth.  
✅ **Dashboard** – Personalized dashboard showing user’s plants and actions.  
✅ **Add & Manage Plants** – Easily add, edit, or delete plant entries.  
✅ **Protected Routes** – Restrict access to sensitive pages for logged-in users only.  
✅ **Responsive Design** – Fully mobile-friendly and visually clean interface.  
✅ **API Integration** – Connected with an Express.js & MongoDB backend for data persistence.

---

## ⚙️ Technology Used

| Technology          | Description                          |
|---------------------|--------------------------------------|
| **React.js**        | Frontend library for building UI     |
| **React Router**    | Page navigation & route protection   |
| **Firebase Auth**   | Authentication system                |
| **Axios**           | HTTP client for API communication    |
| **Tailwind CSS**    | Utility-first CSS framework          |
| **DaisyUI**         | Pre-built Tailwind components        |

## 🧑‍💻 Instructions to Run Locally

Follow these steps to run the client app locally:

```bash
# 1. Clone the repository
git clone https://github.com/arnab44444/Plant-Tracker-Client.git

# 2. Navigate into the project directory
cd Plant-Tracker-Client

# 3. Install project dependencies
npm install

# 4. Add Firebase configuration
# 👉 Create a .env file or configure firebase.config.js

# 5. Start the development server
npm run dev

# 6. Open your browser and go to:
http://localhost:5173
